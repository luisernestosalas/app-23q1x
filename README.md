# CRM App - Gestión de Clientes y Notas

Una aplicación web sencilla para la gestión de clientes y sus notas asociadas, desarrollada con tecnologías web básicas.

## 🚀 Demo

**URL de la aplicación:** [https://app-23q1x-mufa0v2my-luisernestosalas-2775s-projects.vercel.app](https://app-23q1x-mufa0v2my-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- ✅ Gestión completa de clientes (CRUD)
- ✅ Sistema de notas asociadas a cada cliente
- ✅ Interfaz intuitiva y responsiva
- ✅ Almacenamiento local en el navegador
- ✅ Búsqueda y filtrado de clientes
- ✅ Diseño limpio y minimalista

## 🛠️ Stack Tecnológico

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Almacenamiento:** LocalStorage
- **Deployment:** Vercel
