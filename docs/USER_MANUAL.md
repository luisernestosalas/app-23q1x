# Manual de Usuario - CRM de Clientes y Notas

## ¿Qué es esta aplicación?

Es un sistema para gestionar clientes y sus notas. Puedes agregar información de contacto y escribir observaciones sobre cada cliente.

## Acceso a la aplicación

**Dirección web:** https://app-23q1x-mufa0v2my-luisernestosalas-2775s-projects.vercel.app

Solo necesitas un navegador web para usar la aplicación.

## Funciones principales

### 1. Gestión de clientes
- **Agregar clientes nuevos** con información básica
- **Ver lista** de todos los clientes
- **Buscar clientes** específicos
- **Editar información** de clientes existentes
- **Eliminar clientes** cuando sea necesario

### 2. Sistema de notas
- **Crear notas** para cada cliente
- **Ver historial** de notas por cliente
- **Editar notas** existentes
- **Organizar información** de seguimiento

## Cómo usar la aplicación

### Agregar un cliente nuevo
1. Busca el botón **"Nuevo Cliente"** o **"Agregar"**
2. Completa los campos obligatorios
3. Haz clic en **"Guardar"**

### Buscar un cliente
1. Usa la barra de búsqueda en la parte superior
